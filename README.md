# Photo Browser 
A simple Photo Browser that lets you browse photos with random titles and returns the image url via mega cool alert function

---

## Live Demo

https://makekop.github.io/photo-browser/

---

## Local Development

### 1. Clone the repository

```bash
git clone https://github.com/makekop/photo-browser.git
cd photo-browser-vite
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

### 4. Open in browser

Visit:

```
http://localhost:5173/photo-browser/
```
